Crash Engine

Crash Engine is a private iPhone application built with SwiftUI and developed by Code 3 Coders￼.

The application is designed to streamline collision investigation documentation by providing a fast, structured workflow for narrative generation, case organization, photo management, and information exchange collection.

This repository is maintained as a private development project and is not intended for public distribution.

⸻

Purpose

Crash Engine was created to improve efficiency, consistency, and usability during collision investigations and report writing.

The application focuses on:

* Rapid field documentation
* Structured narrative generation
* Consistent reporting workflows
* Efficient information collection
* Practical patrol usability

⸻

Core Features

Narrative Builder

* Guided collision report workflow
* Structured narrative generation
* Additional notes support
* Approximation statement support
* Military time formatting

Photo Management

* Capture photos directly within the app
* Import photos from device photo library
* Persistent photo storage with saved cases

Information Exchange

* Full-screen information entry interface
* Driver information storage
* Vehicle information storage
* Insurance information storage
* Share/export functionality
* Persistent local storage

Case Management

* Local case saving
* Clear form safeguards
* Export/share support
* Stable RTF export behavior

User Interface

* SwiftUI-based design
* Optimized keyboard handling
* Full-screen data entry workflow
* Simplified navigation
* Light mode support
