# Crash Engine Support

Welcome to the official support repository for Crash Engine by Code 3 Coders.

Crash Engine is a mobile collision documentation and information exchange application designed to streamline field reporting workflows with structured reporting tools, photo capture, and organized information exchange features.

---

# Support Requests

For bug reports, feature requests, troubleshooting assistance, or general support, please contact the development team directly through GitHub.

## How to Request Support

### 1. Open a GitHub Issue

Please create a new Issue in this repository and include:

* Device model
* iOS version
* Crash Engine version
* Description of the issue
* Steps to reproduce the issue
* Screenshots if applicable

Providing detailed information helps the development team diagnose and resolve issues more efficiently.

# Current Application Scope

Current public releases of Crash Engine are focused on:

* Property-damage-only collision documentation
* Structured boilerplate narrative generation
* Information exchange workflows
* Photo organization and storage
* Local device data storage

Future updates may expand functionality and reporting capabilities.

---

# Privacy and Data Handling

Crash Engine is designed with a local-first storage philosophy.

* No cloud storage operated by Code 3 Coders
* No AI processing of report contents
* No third-party sale of user data
* User-generated report information remains stored locally on the user device unless intentionally shared by the user

Please review the official Privacy Policy for additional information.
